# Цветочный магазин

Интернет-магазин цветов с авторизацией, корзиной и каталогом товаров. Проект разработан в рамках учебного задания.

## Технологии

- **React** + **TypeScript**
- **Vite** — сборка проекта
- **React Router** — маршрутизация
- **React Hook Form** + **Yup** — валидация форм
- **CSS Modules** — стилизация компонентов
- **Context API** — управление состоянием (авторизация, корзина)

## Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Dana8378/practice-project.git

# 2. Перейти в папку проекта
cd practice-project

# 3. Установить зависимости
npm install

# 4. Запустить проект в режиме разработки
npm run dev

# 5. Открыть в браузере
# http://localhost:5173
```
